## DevOps — donde el código se vuelve producto

Esta sección trata sobre la infraestructura que hace que el software llegue a producción de forma confiable: **CI/CD, contenedores, IaC y automatización operativa.**

No es solo configurar servidores. Es decidir cómo el código pasa de un commit a estar corriendo en producción sin fricciones ni sorpresas.
