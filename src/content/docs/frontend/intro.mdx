## Definición

El **Frontend** (o desarrollo del lado del cliente) es todo lo que los usuarios ven y con lo que interactúan en un sitio web o aplicación. Es la intersección entre diseño, funcionalidad y experiencia de usuario.

### Pilares Fundamentales

1.  **HTML (HyperText Markup Language)**: La estructura y semántica del contenido.
2.  **CSS (Cascading Style Sheets)**: El diseño, layout y presentación visual.
3.  **JavaScript**: La interactividad, lógica y dinamismo.

### El Rol del Desarrollador Frontend

Un desarrollador frontend se encarga de:
-   Traducir diseños a código funcional.
-   Asegurar que la aplicación sea responsiva (funcione en móviles, tablets y escritorio).
-   Optimizar el rendimiento y la accesibilidad.
-   Integrar la interfaz con APIs y servicios del backend.

> "El frontend es la carta de presentación de cualquier producto digital."
